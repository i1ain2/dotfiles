# Setup

## Install chezmoi

### Mac

```shell
$ brew install chezmoi
```

### Arch Linux
```shell
$ sudo pacman -S chezmoi
```

## Apply
### Prerequisite
- node.js (fnm recommended)

### Common
```shell
$ chezmoi init https://github.com/i1ain2/dotfiles.git
$ chezmoi diff
$ chezmoi apply

$ nvim
```
