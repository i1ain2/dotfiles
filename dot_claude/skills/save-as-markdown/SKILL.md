# レポート出力スキル

会話で議論した内容をMarkdownファイルとして保存する。

## デフォルト設定

- **出力先**: `~/src/github.com/i1ain2/notes/reports/`
- **ファイル名形式**: `YYYY-MM-DD-<topic>.md`（日付プレフィックス付き）

## 使用方法

ユーザーが以下のように指示した場合に発動：
- 「mdにして出力して」
- 「レポートとして保存して」
- 「markdownにして書き出して」

## 処理フロー

1. ユーザーが出力先を指定していない場合はデフォルトディレクトリを使用
2. ファイル名が指定されていない場合は会話のトピックから適切な名前を生成
3. 日付プレフィックス（YYYY-MM-DD-）を付与
4. ディレクトリが存在しない場合は作成
5. Markdownファイルを出力
6. 出力パスをユーザーに報告

## 出力フォーマット

```markdown
# タイトル

## 概要
（トピックの要約）

## 詳細
（議論した内容を構造化）

## 参考リンク
（関連URLがあれば）
```

## 例

```bash
# ユーザー: 「AURについての話をmdにして」
# 出力: ~/src/github.com/i1ain2/notes/reports/2026-01-17-aur-security-guide.md
```
