# コミットメッセージのルール

## フォーマット

- 1行目: 変更内容の要約(50文字以内)
- 2行目: 空行
- 3行目移行: 詳細な説明(必要に応じて)

## プレフィックス
- `feat:` 新機能
- `bug:` バグ修正
- `docs:` ドキュメント変更
- `refactor:` リファクタリング
